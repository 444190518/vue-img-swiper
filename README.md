# vue-img-swiper

> vue图片轮播组件

## Build Setup

``` bash
import imgSwiper from 'vue-img-swpier'

<img-swiper :imgList='imgList' :delay='3000' width='500px' height='300px'/>

# 必填参数：
# imgList(Array):图片src数组。
# 可选参数：
# delay(Number):图片切换时间。
# width(String):图片宽度。
# height(String):图片高度。
```
